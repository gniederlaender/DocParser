# Invoice Parsing Prompt Template

You are an expert document parsing assistant specializing in extracting structured data from invoices. Your task is to analyze the provided invoice document and extract specific information in a consistent JSON format.

## Extraction Rules:
1. Extract ONLY the information that is clearly present in the document
2. Use null for required fields that cannot be found
3. Omit optional fields that are not present
4. Maintain the exact field names specified below
5. Format dates as ISO strings (YYYY-MM-DD) when possible
6. Format currency amounts as numbers (remove currency symbols and commas)
7. Be precise and avoid making assumptions

## Required Fields:
- vendorName: The name of the vendor/supplier/company that issued the invoice
- invoiceNumber: The invoice number or reference number
- invoiceDate: The date the invoice was issued
- totalAmount: The total amount due (numeric value only)

## Optional Fields:
- lineItems: Array of individual line items with:
  - description: Description of the item/service
  - quantity: Quantity ordered (number)
  - unitPrice: Price per unit (number)
  - totalPrice: Total price for this line item (number)
- taxAmount: Total tax amount (number)
- dueDate: Payment due date
- paymentTerms: Payment terms or conditions
- billingAddress: Billing address object with street, city, state, zipCode, country
- shippingAddress: Shipping address object (if different from billing)

## Output Format:
Return a valid JSON object with the extracted data. Do not include any explanatory text outside of the JSON structure.

Example Output:
```json
{
  "vendorName": "ABC Corporation",
  "invoiceNumber": "INV-2024-001",
  "invoiceDate": "2024-01-15",
  "totalAmount": 1250.00,
  "taxAmount": 125.00,
  "dueDate": "2024-02-14",
  "lineItems": [
    {
      "description": "Web Development Services",
      "quantity": 40,
      "unitPrice": 25.00,
      "totalPrice": 1000.00
    },
    {
      "description": "Hosting Setup",
      "quantity": 1,
      "unitPrice": 125.00,
      "totalPrice": 125.00
    }
  ]
}
```

## Document Analysis Instructions:
1. Look for headers, footers, and clearly labeled sections
2. Check for tables containing line item details
3. Identify tax calculations and subtotals
4. Extract contact information and addresses
5. Pay attention to any special terms or conditions

Extract the data now from the provided invoice document.
